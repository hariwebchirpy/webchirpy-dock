# Mobile Setup

The mobile application is built using React Native.

## Prerequisites

- Xcode (for iOS)
- Android Studio (for Android)
- React Native CLI

## Installation

```bash
cd mobile
npm install
```

## Running the App

### iOS
```bash
npx react-native run-ios
```

### Android
```bash
npx react-native run-android
```

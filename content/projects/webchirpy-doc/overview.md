# Overview

**WebChirpy Docs** is a premium, developer-focused documentation system designed for speed, clarity, and aesthetics. It serves as our internal knowledge hub, built with the latest web technologies to provide a high-performance documentation experience.

## Key Features

- **Command Palette Search**: Press `⌘K` to instantly search across all projects and pages using our internal indexing API.
- **Interactive TOC**: A dynamic, scroll-synced Table of Contents that tracks your progress through documentation.
- **Micro-Animations**: Fluid, staggered entry animations and page transitions powered by `framer-motion`.
- **Next.js 16 + Tailwind 4**: Built on the absolute cutting edge for maximum performance and future-proofing.
- **Markdown-First**: Clean separation of content and code, supporting GFM and syntax highlighting.
- **True Dark Mode**: A premium "True Black" theme inspired by the official Next.js documentation.

## Tech Stack

- **Framework**: Next.js 16 (App Router)
- **Styling**: Tailwind CSS 4
- **Animation**: Framer Motion 12
- **Components**: Shadcn UI (Radix Primitives)
- **Content**: MDX-lite (Markdown + Gray Matter)
- **Runtime**: Bun

# Frontend Setup

Follow these steps to get the frontend up and running.

## Prerequisites

- Node.js 18+
- npm or yarn

## Installation

```bash
npm install
```

## Running the Dev Server

```bash
npm run dev
```

Open [http://localhost:3000](http://localhost:3000) to see the result.
